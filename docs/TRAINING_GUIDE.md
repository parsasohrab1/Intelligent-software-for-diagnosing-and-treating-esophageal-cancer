# راهنمای آموزش تیم

این راهنمای جامع برای آموزش تیم استفاده از سیستم است.

## ماژول‌های آموزش

### 1. معرفی سیستم (30 دقیقه)

**هدف:** آشنایی کلی با سیستم

**محتوا:**
- معرفی سیستم
- ویژگی‌های اصلی
- نقشه سیستم
- دسترسی‌ها

**پس از تکمیل:**
- درک کلی از سیستم
- آشنایی با رابط کاربری

---

### 2. مدیریت بیماران (45 دقیقه)

**هدف:** یادگیری مدیریت اطلاعات بیماران

**محتوا:**
- افزودن بیمار جدید
- مشاهده اطلاعات
- افزودن داده‌های بالینی
- مدیریت رضایت‌نامه‌ها

**تمرین:**
- افزودن 3 بیمار نمونه
- افزودن داده‌های بالینی

---

### 3. تحلیل تصاویر پزشکی (60 دقیقه)

**هدف:** یادگیری استفاده از سیستم تحلیل تصاویر

**محتوا:**
- آپلود تصاویر
- تحلیل خودکار
- تفسیر نتایج
- استفاده از Explainable AI

**تمرین:**
- آپلود و تحلیل 5 تصویر
- تفسیر Heatmaps

---

### 4. پشتیبانی از تصمیم‌گیری بالینی (45 دقیقه)

**هدف:** یادگیری استفاده از سیستم CDS

**محتوا:**
- پیش‌بینی ریسک
- توصیه‌های درمانی
- پیش‌بینی پاسخ به درمان
- تفسیر نتایج

**تمرین:**
- اجرای پیش‌بینی برای 3 بیمار
- بررسی توصیه‌ها

---

### 5. راهنمای جراحی Real-Time (60 دقیقه)

**هدف:** یادگیری استفاده از سیستم راهنمای جراحی

**محتوا:**
- اتصال به endoscope
- تفسیر Overlay
- استفاده از حاشیه امن
- نکات ایمنی

**تمرین:**
- شبیه‌سازی استفاده در عمل
- تفسیر نتایج

---

### 6. امنیت و حریم خصوصی (30 دقیقه)

**هدف:** آشنایی با اصول امنیت

**محتوا:**
- کنترل دسترسی
- رمزگذاری
- مدیریت رضایت‌نامه
- انطباق با استانداردها

---

### 7. استفاده از API (90 دقیقه)

**هدف:** آموزش برای توسعه‌دهندگان

**محتوا:**
- احراز هویت
- Endpoints اصلی
- مثال‌های کد
- یکپارچه‌سازی

---

## نحوه استفاده از سیستم آموزش

### ثبت‌نام در ماژول

1. رفتن به "آموزش" → "ماژول‌ها"
2. انتخاب ماژول
3. کلیک روی "ثبت‌نام"

### تکمیل آموزش

1. مطالعه محتوا
2. انجام تمرین‌ها
3. به‌روزرسانی پیشرفت
4. تکمیل آزمون (در صورت وجود)

### دریافت گواهینامه

پس از تکمیل ماژول و گذراندن آزمون، گواهینامه صادر می‌شود.

4. امنیت و حریم خصوصی

1. معرفی سیستم
2. مدیریت بیماران
3. امنیت و حریم خصوصی

### برای توسعه‌دهندگان

1. معرفی سیستم
2. استفاده از API
3. یکپارچه‌سازی

## آزمون‌ها

### نکات آزمون

- زمان محدود
- امکان تکرار
- نمایش پاسخ‌های صحیح پس از آزمون
